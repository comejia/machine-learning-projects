# Machine Learning

Este repositorio contiene ejemplos prácticos de algoritmos de Machine Learning utilizando Python y bibliotecas populares como Scikit-Learn, Pandas y NumPy. Cada ejemplo está documentado y explicado paso a paso para facilitar el aprendizaje y la comprensión de los conceptos clave.

## Contenido
- [Regresión](./regression): Ejemplos de regresión lineal y regresión logística.
- [Clasificación](./classification): Ejemplos de clasificación utilizando árboles de decisión, SVM y KNN.
- [Clustering](./clustering): Ejemplos de clustering con K-Means y DBSCAN.
- [Reducción de Dimensionalidad](./dimensionality_reduction): Ejemplos de PCA y t-SNE.
- [Validación de Modelos](./model_validation): Técnicas de validación cruzada y evaluación de modelos.
- [Preprocesamiento de Datos](./data_preprocessing): Técnicas de limpieza y transformación de datos.
- [Series Temporales](./time_series): Análisis y predicción de series temporales.
- [Redes Neuronales](./neural_networks): Introducción a redes neuronales con TensorFlow y Keras.
